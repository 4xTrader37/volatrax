# **App Name**: Volatrax Digital Services

## Core Features:

- Home/About Page: Display introductory content, service overview, and a 'Why Choose Us' section.
- Account Management Details: Display details and rules for 'Small Account Management' and 'Big Account Management' services.
- Course Listings: Fetch and display available courses from Firestore, each with a title, description, price, and WhatsApp contact button.
- Course Data Management: Store and manage course information (title, description, price, image, refund text) in Firestore.
- WhatsApp Contact Button: Implement a global floating WhatsApp button for direct customer support with a pre-filled message option for each course.
- Contact Via WhatsApp AI Assistant: AI powered tool for pre-generating contact via WhatsApp prefilled messages with context based on what the user has browsed and added to cart.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a trustworthy and professional feel.
- Background color: Light gray (#F0F4F8), a muted version of the primary color.
- Accent color: Soft gold (#D4AF37), for highlighting key information and CTAs.
- Body and headline font: 'Inter', a sans-serif for a modern, neutral appearance.
- Use simple, professional icons related to digital services, trading, and education.
- Design a clean, modern, and responsive layout suitable for mobile and desktop viewing.
- Use subtle animations for transitions and interactions to enhance user experience without being distracting.
